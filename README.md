# AfriStay

A vehicle and real estate rental platform for Rwanda.

## Setup
1. Create a Supabase project.
2. Run the SQL Schema provided in the tutorial.
3. Update `assets/js/config.js` with your URL and Key.
4. Upload all files to Cloudflare Pages / GitHub.

## Folder Structure
- `assets/css/` : Styles
- `assets/js/` : Logic
- `auth.html` : Login/Signup
- `contact.html` : Contact Form (Connects to DB)
